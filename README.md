# 🌿 ISEYAA TiarionX — Ogun State Geo-Intelligent OS (Enterprise)

ISEYAA is the world's most advanced, AI-powered **State Operating System**, integrating Tourism, Culture, and Governance into a scalable digital hub for all 20 LGAs in Ogun State, Nigeria.

---

## 🏛️ Repository Architecture (Turborepo)
Managed by a **20-person squad structure**, this monorepo is partitioned into high-performance workspaces:

### 🚀 Apps (`/apps`)
- **`web`**: The main user portal (Next.js 14+).
- **`api`**: The TiarionX Intelligence Engine (FastAPI).
- **`dashboard`**: Government Economic Command Center (In-Dev).

### 📦 Packages (`/packages`)
- **`ui-kit`**: Shared Design System (Squad Beta).
- **`database`**: Supabase/PostgreSQL schema (Squad Alpha).
- **`ai-agents`**: The Agentic Brain (Orchestrator, Crawler, Planner).
- **`config-eslint`**: Standardized linting rules.

---

## 👥 Team Operations
This repository supports **4 specialized squads**:
- **Squad Alpha (Core/AI):** Backend, DB, Prompt Engineering.
- **Squad Beta (Experience):** Frontend, UI/UX, Design.
- **Squad Gamma (Integrations):** LGA Intelligence, Maps, APIs.
- **Squad Delta (DevOps/QA):** Testing, Pipelines, Infrastructure.

See [SQUADS.md](project-management/SQUADS.md) for current team partitioning.

## 🚦 Quick Start (Local Setup)

1. **Clone the Repo:**
   ```bash
   git clone https://github.com/LJENTERTAINMENTMUSIC/iseyaa-tiarionx.git
   ```

2. **Install Dependencies:** (pnpm required for Monorepo)
   ```bash
   pnpm install
   ```

3. **Start the Dev Environment:** (Turbo-cached)
   ```bash
   pnpm dev
   ```

4. **Contributing:** Read the [CONTRIBUTING.md](CONTRIBUTING.md) before pushing any code.
